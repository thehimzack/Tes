Installation

    IMPORTANT: You must update from version 1.4, if you upgrade from another version will throw an error.
  • Upload the file v1.5.zip found inside the Update-v1.5 folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Unzip the file v1.5.zip and enter at URL https://yoursite.com/v1.5/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v1.5/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.

  • Add this new text string in your file resources / lang / en / general.php
    • // Version 1.5
'subscription_price_required' => 'Subscription Price field is required.',
    If you have multiple languages on your site you should add to each file.
