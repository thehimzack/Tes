Installation

    IMPORTANT: If you have made changes to the script make a backup of the files, because they can be replaced.
    IMPORTANT: You must update from version 1.6, or version 1.7. if you upgrade from another version will throw an error.
  • Upload the file v1.8.zip found inside the Update-v1.8 folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Unzip the file v1.8.zip and enter at URL https://yoursite.com/v1.8/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v1.8/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.
