Installation

    IMPORTANT: You must update from version 1.1, if you upgrade from another version will throw an error.
  • Upload the file v1.2.zip found inside the Update-v1.2 folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Unzip the file v1.2.zip and enter at URL https://yoursite.com/v1.2/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v1.2/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.

  • Add this new text string in your file resources / lang / en / general.php
    • // Version 1.2
'bank' => 'Bank',
'pinned_post' => 'Pinned Post',
'pin_to_your_profile' => 'Pin to your profile',
'post_pinned_success' => 'Post pinned successfully!',
'post_unpinned_success' => 'Post has been unpinned successfully',
'unpin_from_profile' => 'Unpin from profile',
'show_widget_creators' => 'Show widget Creators Featured on the homepage',
'dark_mode' => 'Dark mode',
'light_mode' => 'Light mode',
'bookmarks' => 'Bookmarks',
'desc_bookmarks' => "Posts you've saved",
'no_bookmarks' => 'No bookmarks yet',
    If you have multiple languages on your site you should add to each file.
