Installation

    IMPORTANT: If you have made changes to the script make a backup of the files, because they can be replaced.
    IMPORTANT: You must update from version 2.8. if you upgrade from another version will throw an error.
    IMPORTANT: Please follow the steps below strictly, otherwise the update will fail.
  • Upload the file v2.9.zip found inside the Update-v2.9 folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Unzip the file v2.9.zip and enter at URL https://yoursite.com/v2.9/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v2.9/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.
