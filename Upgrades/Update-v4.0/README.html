Installation

    IMPORTANT: If you have made changes to the script make a backup of the files, because they can be replaced.
    IMPORTANT: You must update from version 3.9. if you upgrade from another version will throw an error.
    IMPORTANT: Please follow the steps below strictly, otherwise the update will fail.
  • Upload the folder vendor, the files composer.json and composer.lock found inside the Script folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory (You can upload the vendor folder from FTP or by compressing and then uploading to the server).

  • Upload the file v4.0.zip found inside the Update-v4.0 folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Unzip the file v4.0.zip and enter at URL https://yoursite.com/v4.0/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v4.0/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.
