Installation

    IMPORTANT: If you have made changes to the script make a backup of the files, because they can be replaced.
    IMPORTANT: You must update from version 2.1. if you upgrade from another version will throw an error.
  • Upload the file v2.2.zip found inside the Update-v2.2 folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Unzip the file v2.2.zip and enter at URL https://yoursite.com/v2.2/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v2.2/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed. (Maybe the session has been closed due to the change of driver of the sessions).
