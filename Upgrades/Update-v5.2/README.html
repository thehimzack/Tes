Installation

    IMPORTANT: If you have made changes to the script make a backup of the files, because they can be replaced.
    IMPORTANT: You must update from version 5.1. if you upgrade from another version will throw an error.
    IMPORTANT: Please follow the steps below strictly, otherwise the update will fail.
  • Upload the file v5.2.zip found inside the Upgrades > Update-v5.2 folder, into the public_html or www folder on your server, or where you have the script installed, you must make sure that it is the root directory.

  • Unzip the file v5.2.zip and enter at URL https://yoursite.com/v5.2/ or if you have the script in subdirectory (subdomain) https://yoursite.com/test/v5.2/
    • IMPORTANT: You must be logged in as Admin

  • If all goes well, this message will appear, that's it! The new version has already been installed.
    • IMPORTANT: If you are using Coconut® and use a region other than North Virginia (USA) you must configure in Dashboard > Video encoding
